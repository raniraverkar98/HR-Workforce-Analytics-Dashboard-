An interactive Power BI HR Workforce Analytics Dashboard designed to transform raw HR data into meaningful business insights. This project provides a comprehensive view of key workforce metrics, enabling data-driven decision-making for HR professionals and business leaders.

Key Features
📊 Interactive and dynamic Power BI dashboards
👥 Employee demographics and workforce distribution
📈 Attrition and retention analysis
💼 Department and job role insights
⭐ Job satisfaction and performance metrics
🎯 KPI cards and drill-through analysis
🎛️ Slicers and filters for customized reporting
Tools & Technologies
Power BI
Power Query
DAX
Excel (Dataset)

Purpose

The purpose of this project is to analyze workforce data, identify employee trends, monitor key HR metrics, and support data-driven decision-making. It helps HR professionals gain insights into attrition, employee demographics, job satisfaction, performance, and overall workforce health.
